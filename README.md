# POGuy: nice guy open your project workspace

## Config

Place your `.poguy` in your home directory.

Example:

```json
{
	"projects": [
		{
			"name": "poguy",
			"directory": "/Users/kyktommy/test",
			"programs": [
				{
					"name": "Open in finder",
					"execute": "open ."
				}
			]
		}
	]
}
```
